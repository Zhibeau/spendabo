# Test Data for Multimodal Import Testing

Place your test files in this directory to test PDF and image imports.

## Supported File Types

### Images
- **JPG/JPEG** - Receipt photos
- **PNG** - Screenshots of transactions
- **WebP** - Modern image format
- **HEIC/HEIF** - iPhone photos

### Documents
- **PDF** - Bank statements, credit card statements
- **CSV** - Traditional CSV exports

## Sample Test Files

You can add test files like:
- `receipt.jpg` - A photo of a paper receipt
- `bank-statement.pdf` - A PDF bank statement
- `credit-card.pdf` - A credit card statement
- `screenshot.png` - A screenshot of online transactions
- `transactions.csv` - A CSV export from your bank
